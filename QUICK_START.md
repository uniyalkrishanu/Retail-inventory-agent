# Quick Start Guide

## First Time Setup

### Step 1: Install Docker Desktop
1. Download Docker Desktop from: https://www.docker.com/products/docker-desktop
2. Run the installer
3. Restart your computer when prompted

### Step 2: Launch the Application
1. Double-click **start.bat**
2. Wait for the setup to complete (first run takes 2-5 minutes)
3. Your browser will open automatically to the application

## Daily Usage

- **Start**: Double-click `start.bat`
- **Stop**: Double-click `stop.bat`
- **Access**: Open http://localhost:3000 in your browser

## Troubleshooting

| Problem | Solution |
|---------|----------|
| "Docker is not installed" | Install Docker Desktop and restart |
| "Docker is not running" | Wait for Docker to start, or open Docker Desktop manually |
| "Failed to start" | Run `stop.bat`, then `start.bat` again |
| Page won't load | Wait 30 seconds for containers to initialize |
